# Custom JPEG
Encode and decode a file using the custom jpeg algorithm


### Install

```
    $ pip install -e .
```
  
### Using

Examples:

Encode a file using the algorithm:

```
    cjpeg <file> [FLAGS]
```
    
Decode a file using the algorithm:

```
    cjpeg <file>.cjpeg [FLAGS]
```

More options:

```
    $ cjpeg --help
```
