# Custom JPEG
Encode a picture using the custom jpeg algorithm


### Install

```
    $ pip install -e . --process-dependency-links
```
  
### Using

Examples:

Encode a file using the algorithm:

```
    cjpeg <file> [FLAGS]
```
    
Decode a file using the algorithm:

```
    cjpeg <file>.cjpeg [FLAGS]
```

More options:

```
    $ cjpeg --help
```
